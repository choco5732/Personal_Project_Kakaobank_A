<H1>📽️ 프로젝트</H1>
<b>Searcher</b>**앱은 Retrofit과 카카오API를 이용한 이미지 검색 앱입니다**.<br> 



      
<H1>🎯 기술 스택</H1>

|제목|내용|
|------|---|
|Language|Kotlin|
|jetpack|AAC ViewModel, LiveData, ViewBinding|
|network|Retrofit2, OkHttp3, Coroutine|
|data|SharedPreference|
|ImageLoading|Glide|

<H1> 🎄 Architecture</H1>
- `MVVM` `CleanArchitecture`
